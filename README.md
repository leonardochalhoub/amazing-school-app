## 🧭 Why this exists

English schools still sell the same thing they sold twenty years ago: a printed book, a whiteboard, and a teacher who works 60 hours a week to keep up.

**Amazing School is an open-source operating system for the modern English teacher.** We take the grunt work off the teacher's shoulders — lesson generation, progress tracking, assignment routing, tuition control — and leave them with what only a human can do: notice a student's face fall when they say *"I'm not good at past simple"* and turn it into a teaching moment.

> *"The speed of AI, the judgment of a teacher, in the same classroom."*

&nbsp;

## 🧑‍🏫 What this software is for — autonomous English teachers

Amazing School is built for one specific person: the **independent English teacher** who is simultaneously the lesson planner, the homework reviewer, the billing department, the marketing team, the tech support, and the person actually teaching the class. Before v1.0.0, that person used a WhatsApp group, a spreadsheet, three different video-call links, and maybe a Moodle instance from 2019.

With v1.0.0, one self-hosted instance of Amazing School covers the whole operating loop:

- **📚 Ready-made curriculum.** 250+ lessons already authored across CEFR A1 → C1, reviewable and editable before publishing. No more copy-pasting from six textbooks.
- **🎤 Pronunciation feedback at scale.** Every student can drill speaking a hundred times a week with a 15-probe Whisper ensemble — the teacher only hears about the hard cases.
- **🗓️ Scheduling + attendance.** Create a Google Meet session, mark the class Done / Rescheduled / Absent, and it flows into the student's history automatically.
- **💰 Tuition as a first-class feature.** Set a per-student monthly amount, see the full matrix from 2024 to today, tap a cell when the payment lands. Debt and discounts are built in; no teacher should need a second spreadsheet for this.
- **📓 Diary + history as memory prosthesis.** One teacher can remember 20 students — but not every mood, every grammar slip, every promise made in the last session. The diary and student-history tables are the teacher's long-term memory.
- **🎓 A dashboard the student actually wants to visit.** Streaks, XP, badges, leaderboard, assigned lessons — the gamification loop is what keeps adults practicing between sessions.
- **🤖 An AI tutor that never sleeps.** Students can ask Claude at 11pm on a Sunday without your phone buzzing. Every conversation is logged and visible to you the next day if you want to read it.
- **🛡️ Privacy that a teacher can defend to a parent.** Each teacher's data is walled off from the next. The sysadmin dashboard only sees aggregate platform counts — tuition, chat content, and diary entries are invisible to everyone except the teacher who owns them.

If you're running a one-person online school — or thinking about becoming one — this is the toolbox.

## 🏗️ Architecture

```
┌───────────────────────────────┐     ┌──────────────────────┐     ┌─────────────────────┐
│  Next.js 16 + React 19        │     │   Supabase           │     │   Claude API        │
│  App Router · Server Actions  │ ──► │   Postgres + Auth    │     │   Haiku 4.5 (chat)  │
│  Tailwind v4 · shadcn/ui      │     │   Storage · Realtime │     │   Sonnet 4.6 (gen)  │
│  Turbopack · Recharts         │     │   Row-Level Security │     │                     │
└───────────────────────────────┘     └──────────────────────┘     └─────────────────────┘
             │                                   │                           │
             ▼                                   ▼                           ▼
   ┌──────────────────┐              ┌──────────────────┐         ┌─────────────────────┐
   │  Groq Whisper    │              │  Vercel (edge)   │         │  Offline pipeline   │
   │  pronunciation   │              │  CDN + SSR       │         │  content authoring  │
   └──────────────────┘              └──────────────────┘         └─────────────────────┘
```

**The Ledger** → Supabase Postgres. Every student action — XP earned, lesson completed, assignment skipped, chat turn, payment received — lives here, RLS-guarded.

**The Tutor** → Claude Haiku 4.5 for live student chat. Claude Sonnet 4.6 for offline lesson authoring. Groq Whisper for pronunciation scoring with a 15-probe ensemble and adversarial phonetic prompting.

**The Catalog** → Static JSON (`content/lessons`, `content/music`). Content is versioned in git and loaded at request time — no DB round-trip, no latency tax.

| **AI content** | Claude Sonnet 4.6, Google Gemini 2.5 Flash | Batched offline, cost-capped |
| **Pronunciation** | Groq Whisper (large-v3 · turbo · distil) | 15-probe majority-voting ensemble |
| **Testing** | Vitest + Playwright | Unit, integration, e2e |
| **Hosting** | Vercel | 100 GB bandwidth free tier |

## 📐 Design principles

- **Zero subscriptions forever.** 100% free, MIT-licensed, no dark patterns.
- **Teacher-in-the-loop, always.** AI drafts. Humans ship.
- **Privacy by absence.** The admin dashboard is coded to *never* query per-student tuition, chat content, or any teacher-to-student revenue relationship. If it's not in the query, it can't leak.
- **Brazilian first.** Every error string, lesson title, and tooltip is bilingual (PT-BR / EN) from day one.
- **No premature abstractions.** Three similar lines in three files is better than a cleverness tax nobody asked for.

## 🧪 Highlights shipped in v1.0.0

- 🎯 **Pronunciation scoring rebuilt** — 15 parallel Groq probes, phonetic adversarial prompts, duration anomaly detection, word-level clarity, majority-voting rescue penalty.
- 📈 **Diablo-style XP curve** — `50n² + 50n` per level. Level 1→2 costs 100 XP; Level 10→11 costs 5,500. Feels like leveling up.
- 🏅 **Futuristic badge system** — 14 badges, rarity tiers (Common → Mythic), neon gradients with per-badge glow. *First Contact* is awarded the moment you join.
- 🔴 **Teacher presence indicator** — glowing-green pill with a breathing pulse animation when you're online.
- 🪟 **Live demo** — 2+ years of seeded data reaching back to January 2024. Ana's birthday is pinned to tomorrow *forever* so the reminder panel is always populated.
- 🛡️ **Privacy-scoped sysadmin** — aggregate counts only. Per-student tuition, chat content, and teacher finances never reach the admin UI.
- 🔒 **Honest privacy notice** — PT-BR + EN disclosure on every profile, listing exactly what we store and what we never do.
- 🎵 **Music catalog** — every song has a YouTube ID, 30 exercises, and an auto-generated Letras.mus.br translation link.

## 🚀 Run it locally

```bash
git clone https://github.com/leonardochalhoub/amazing-school-app-1.git
cd amazing-school-app-1
cp .env.example .env.local
# Fill NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY,
# SUPABASE_SERVICE_ROLE_KEY, ANTHROPIC_API_KEY

npm install
npm run dev                 # http://localhost:3000

# Optional: seed the demo teacher + 13 students
npm run seed:demo
```

### Handy scripts

| Script | What it does |
|---|---|
| `npm run dev` | Local dev server (Turbopack) |
| `npm run build` | Production build |
